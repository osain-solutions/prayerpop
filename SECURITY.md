# Security Policy

## Supported Versions

Security updates are provided for the latest released version of PrayerPop.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing support@osain.ee.

Include the affected version, a clear description of the issue, steps to reproduce it, and any relevant proof of concept. Please do not open a public GitHub issue for security reports.

We will acknowledge valid reports as soon as possible and coordinate a fix before public disclosure.
